## 📚 Estructura del Curso

### Módulo 00: Nivelación Python
Fundamentos de programación Python aplicados a finanzas
- Introducción a Python
- Estructuras de Control
- Funciones y Módulos
- Estructuras de Datos
- Pandas para Finanzas
- Visualización de Datos
- Python Aplicado a Finanzas

### Módulo 01: Fundamentos
Conceptos básicos de matemática financiera
- Valor del Dinero en el Tiempo
- Tasas de Interés y Conversiones
- Anualidades y Amortización
- Bonos y Valuación
- Optimización de Portafolios

### Módulo 02: Instrumentos
Análisis de instrumentos financieros argentinos
- Análisis de Bonos Argentinos
- Análisis de Acciones Argentinas

### Módulo 03: Análisis Financiero
Análisis de estados financieros
- Análisis de Ratios Financieros
- Análisis Comparativo de Empresas

### Módulo 04: Series Temporales
Modelos predictivos para finanzas
- Predicción de Precios de Acciones
- Predicción de Series Temporales

### Módulo 05: Analítica Agéntica
Integración de IA en análisis financiero
- Análisis Agéntico de Estados Contables
- Agentes de IA para Finanzas
- Asesor Financiero con IA

---

## 🗂️ Datasets Incluidos

- **Empresas Argentinas**: Datos de mercado de acciones
- **Balances Oficiales**: Estados contables de empresas argentinas
- **Libros de Referencia**: Material bibliográfico (Dumrauf)
